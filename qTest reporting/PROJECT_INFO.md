# qTest Reporting - Standalone Project

This is a clean, standalone reporting tool extracted from the main qTest Integration project.

## What's Included

### Core Files
- **src/report.ts** - Main reporting script (generates reports by user and date)
- **src/qtest-client.ts** - Simplified qTest API client (reporting methods only)
- **src/auth.ts** - qTest authentication (OAuth and Bearer Token)
- **src/config.ts** - Configuration loading and validation
- **src/types.ts** - TypeScript type definitions

### Configuration
- **config.example.json** - Example configuration file
- **config.json** - Your actual configuration (gitignored)
- **.gitignore** - Excludes node_modules, dist, output, and config.json

### Documentation
- **README.md** - Comprehensive documentation
- **QUICKSTART.md** - Quick start guide for new users
- **PROJECT_INFO.md** - This file

### Build System
- **package.json** - NPM scripts and dependencies
- **tsconfig.json** - TypeScript configuration
- **dist/** - Compiled JavaScript (generated by `npm run build`)
- **output/** - Generated reports (created automatically)

## Differences from Main Project

This standalone project contains **only** the reporting functionality:

✅ **Included:**
- qTest API querying
- Report generation (console, JSON, CSV)
- Authentication (OAuth and Bearer Token)
- User and project management
- Test execution data retrieval

❌ **Not Included:**
- Webhook server
- Event generation
- File management
- Webhook registration
- Integration-specific logic

## Quick Commands

```bash
# Install
npm install

# Configure
cp config.example.json config.json
# Edit config.json with your credentials

# Run
npm run report                    # Last 7 days
npm run report -- --days 30       # Last 30 days
npm run report -- --all           # All time
npm run report -- --project 1636  # Specific project

# Build
npm run build
npm run report:prod               # Run compiled version
```

## File Size

The standalone reporting project is minimal and lightweight:
- **Source**: ~28 KB (5 TypeScript files)
- **Dependencies**: 43 packages (minimal: axios, typescript, ts-node)
- **Compiled**: ~22 KB JavaScript + source maps

## Distribution

This project is ready to distribute as a standalone tool. You can:
1. Share the entire `qTest reporting` folder
2. Package it as an npm package
3. Create a Docker container
4. Build executables with pkg or nexe

## Maintenance

This project is completely independent from the main qTest Integration project:
- Has its own dependencies
- Has its own configuration
- Can be updated independently
- No shared state or coupling
